# Әжеміздің 70 жасы — мерейтойлық шақыру 🎉

Бір беттік статикалық сайт (index.html). Бұл онлайн-пригласительный.

## Не бар
- Кері санақ (тойға дейінгі күн/сағат/минут)
- «Күнтізбеге қосу» (.ics)
- 2GIS картасына сілтеме
- WhatsApp-пен бөлісу
- Адаптивті дизайн (телефон/компьютер)

## Қалай өзгерту керек
`index.html` ішінен мынаны түзетіңіз:
- **Мейрамхана атауы / нақты мекен-жай** — `[мейрамхана атауын жазыңыз]` деген жерді.
- **Карта сілтемесі** — `mapBtn` href-ін нақты мекенжайға ауыстырыңыз.
- **Шақырушылар** — «Балалары мен немерелері» жолын.
- **Аты** — қаласаңыз «Әжеміздің» дегеннің орнына әженің атын қойыңыз.

Күні (30.11.2026, 19:00) кері санақта да, .ics-те де қойылған.

## Қалай деплой жасау керек (тегін)

### 1) Netlify Drop (ең оңай, 1 минут)
1. https://app.netlify.com/drop ашыңыз
2. `aje-70-shaqyru` папкасын браузерге сүйреп тастаңыз
3. Дайын сілтемені WhatsApp-қа жіберіңіз

### 2) Vercel
```bash
npm i -g vercel
cd aje-70-shaqyru
vercel
```

### 3) GitHub Pages
1. Жаңа репозиторий жасап, осы файлдарды жүктеңіз
2. Settings → Pages → Branch: main → Save
3. `https://<username>.github.io/<repo>/` ашылады
